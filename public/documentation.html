API Documentation

This documentation provides details about the API endpoints available in this application.

1. Root Endpoint

Description: Returns a welcome message when accessing the root of the application.

Endpoint URL: /

Method Type: GET

Query Parameters: None

Request Body: None

Response Format:

Welcome to my app!

2. Get All Users

Description: Returns a list of all users.

Endpoint URL: /users

Method Type: GET

Query Parameters: None

Request Body: None

Response Format: A JSON array of user objects.

[
  {
    "Username": "User1",
    "Email": "user1@example.com",
    "FavoriteMovies": ["MovieID1", "MovieID2"]
  },
  // other users...
]

3. Get a User by Username

Description: Returns data about a single user by their username.

Endpoint URL: /users/:Username

Method Type: GET

Query Parameters: Username (String)

Request Body: None

Response Format: A JSON object with the user's details.

{
  "Username": "User1",
  "Email": "user1@example.com",
  "FavoriteMovies": ["MovieID1", "MovieID2"]
}

4. Add a New User

Description: Creates a new user and adds them to the database.

Endpoint URL: /users

Method Type: POST

Request Body:

{
  "Username": "User1",
  "Password": "password123",
  "Email": "user1@example.com",
  "Birthday": "1990-01-01"
}

Response Format: Returns the newly created user object.

{
  "Username": "User1",
  "Email": "user1@example.com",
  "FavoriteMovies": []
}
